yap-sum

One-click YouTube summaries through your own LLM key. Nothing leaves your browser except the transcript you send to the endpoint you choose.

Preset endpoints
  • OpenAI — https://api.openai.com/v1 · gpt-4o-mini
  • GLM / Z.ai — https://api.z.ai/api/paas/v4 · glm-4.7-flash (free tier)
  • Gemini — https://generativelanguage.googleapis.com/v1beta/openai · gemini-2.5-flash
  • OpenRouter — https://openrouter.ai/api/v1 · any model
  • Groq — https://api.groq.com/openai/v1
  • Ollama (local) — http://localhost:11434/v1 · llama3.2
  • LM Studio (local) — http://localhost:1234/v1
  • Anthropic — provider = Anthropic · https://api.anthropic.com/v1 · claude-fable-5

Local endpoints and non-listed hosts need host permission — yap-sum will request it the first time you save such a URL.
